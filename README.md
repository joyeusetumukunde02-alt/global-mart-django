# global-mart-django

Global Mart is a Django-based e-commerce web application developed as a **group project** for school.  
It simulates a simple online store with user accounts, product listings, and shopping cart functionality.

## My Contribution
- Developed **user authentication** and profile management
- Implemented **shopping cart functionality** and basic order management
- Designed **templates for product and checkout pages**
- Connected models with database for product and user data
